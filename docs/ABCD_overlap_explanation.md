# ABCD 重なり判定の本質（0.49/0.50/0.51）

あなたの定義（θ=π）では

- `Dx = 2R - 1`
- `Dy = 0`
- 原点までの距離 `dist = sqrt(Dx^2 + Dy^2) = |2R-1|`

したがって、**厳密重なり（Dが原点と一致）**は

- `Dx = 0` かつ `Dy = 0`
- つまり `2R - 1 = 0`
- よって **R = 0.5 のときのみ**

となります。

## 0.49 と 0.51 が「見た目で」重なる理由

- `R=0.49` だと `Dx=-0.02`
- `R=0.51` だと `Dx=+0.02`
- 距離はどちらも `0.02`

プロットの拡大率・点サイズ・アンチエイリアス次第で、どちらも「0にいるように見える」ことがあります。
ただし数値はゼロではないので、**厳密には零点通過ではありません**。

## 0.1〜0.9での直感

- `R=0.1` → `Dx=-0.8`
- `R=0.2` → `Dx=-0.6`
- ...
- `R=0.5` → `Dx=0`
- ...
- `R=0.9` → `Dx=+0.8`

`R` が 0.5 から離れるほど `|2R-1|` は大きくなり、原点から離れます。

## あなたの言い方との対応

「Bが0.49ならAは0.51の対称関係」
- A, B の対称は `A_x=-R`, `B_x=R`
- D側のズレ量は `2R-1`
- ズレがゼロになるのは `R=0.5` の一点のみ

この一点性を CSV で記録して示すのが、最も強い説明です。
